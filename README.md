# kytea-java
## Java wrapper of KyTea (http://www.phontron.com/kytea/)

#### Quick Start
1. Install KyTea
  - Mac OS X: `brew install kytea`
  - Ubuntu: `apt-get install kytea`
1. Install SWIG
  - Mac OS X: `brew install swig`
  - Ubuntu: `apt-get install swig`
1. Build Java library and test it
```
    make
    make test
```
